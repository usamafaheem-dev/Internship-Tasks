# React Interactive App

This is a simple React project that demonstrates basic concepts of state management and event handling.

## Features

- Counter with increment and decrement buttons  
- Text input with real-time state update  
- Add and delete items from a list  
- Editable user object (name and age)  
- Proper use of React useState and immutable updates  

## Technologies Used

- React.js  
- JavaScript (ES6)  
- useState Hook  

## How to Run

```bash
npm install
npm start
